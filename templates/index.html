Accepted upload file templates

To ensure high data quality, only strictly formatted files are allowed for uploading. These formats can be ensured using an upload file template for filling out meta data. If you do not have a template, you might be able to find it below. Otherwise contact the database administrator for help (magnus.johannsen@sund.ku.dk).

NOTE: The below templates are updated automatically as changes are made to the database. So make sure to download a new one everytime you need to fill out meta data, or it might not be compatible with the database.
