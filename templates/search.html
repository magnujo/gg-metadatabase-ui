Download All Individual Tables

Press the "Download all tables..." button below to download all raw database tables, including both old unstandardized CGG tables and new standardized field sample data.

Note: You might need to convert data to a common case to avoid case sensitivity issues if you cannot do case-insensitive filtering.

Use UTF-8 if in doubt

NOTE: The button will be disabled when clicked. Update the page to re-enable it

Download Joined Standardized Data (full outer joined)

Press the "Download merged table..." button below to download a merged table, with "new" standardized field sample data (old CGG data not included).

Note: You might need to convert data to a common case to avoid case sensitivity issues if you cannot do case-insensitive filtering.

Use UTF-8 if in doubt

NOTE: If a library was sequenced multiple times, only the QC from the latest sequencing will be in the data. If you need data from other sequencing runs, you can find it in the data folders on dandy: projects/caeg/data/production
