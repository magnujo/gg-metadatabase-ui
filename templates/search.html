Download All SMDB Tables

Press the "Download all tables..." button below to download all raw database tables, including both old unstandardized CGG tables and new standardized field sample data.

Note: You might need to convert data to a common case to avoid case sensitivity issues if you cannot do case-insensitive filtering.

Use UTF-8 if in doubt

NOTE: The button will be disabled when clicked. Update the page to re-enable it


To download the same using curl, use the following command in your terminal: 
curl -L -o data.zip -X POST -d 'encoding_type=utf_8' http://dandyweb01fl.unicph.domain:5100/download_all_individual_tables
Set encoding_type=utf_8 or encoding_type=utf_16 depending on which data you want to include.

Download Joined Standardized SMDB Data (outer joined)

Select which databases you want to include in the download and then press the "Download joined data..." button below to download a zip file with the selected data included.

Note: You might need to convert data to a common case to avoid case sensitivity issues.

WARNING: The result will contain cross-joins between the flowcell table and edna_wetlab_report table. This means that some data in the seqc_tube_tag column(s) will be incorrect. This is due to missing seqc_tube_tag information in the flowcell table. We are working on fixing the issue.

NOTE: A library can be sequenced on multiple flowcells, resulting in different QC reports. To distinguish between reports, use the binf_qc_report_id or binf_qc_report_path column.

The button will be disabled when clicked. Update the page to re-enable it


To download the same using curl, use the following command in your terminal: 
curl -L -o data.zip -X POST -d 'checkbox_smdb=on' -d 'checkbox_qc=on' http://dandyweb01fl.unicph.domain:5100/download_merged_standardized
Set checkbox_smdb=on and/or checkbox_qc=on depending on which data you want to include.
